# Support

Kestrel Desktop is the flagship product surface of the Kestrel Suite. Use the public GitHub repo as the primary support and contribution channel.

## Get Help

- Bugs: open a GitHub Issue with reproduction details and validation context.
- Feature requests: open a GitHub Issue describing the user problem and desired behavior.
- Usage questions: use GitHub Discussions if enabled for the public repo; otherwise open a GitHub Issue.

## Before You File

- Read [README.md](https://github.com/LumiCorp/kestrel/blob/main/README.md) for the Desktop-first product overview.
- Check [CONTRIBUTING.md](https://github.com/LumiCorp/kestrel/blob/main/CONTRIBUTING.md) for setup and validation expectations.
- Review [SECURITY.md](https://github.com/LumiCorp/kestrel/blob/main/SECURITY.md) if the issue might expose a vulnerability or sensitive trust-boundary problem.

## Security

Do not report vulnerabilities in public issues. Use the private reporting path described in [SECURITY.md](https://github.com/LumiCorp/kestrel/blob/main/SECURITY.md).
