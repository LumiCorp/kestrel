# README Branding Design

## Goal

Rebrand the root README around Kestrel One as Lumi Corp's application platform.

## Approved Direction

Use a bold terminal-banner treatment at the top of the README:

- a boxed ASCII header for `Kestrel One`
- a subtitle line for `Kestrel One by Lumi Corp`
- opening copy that describes the repo as Kestrel One rather than a generic app template

## Why

This keeps the README aligned with the product identity while preserving the repo's terminal-first tone.
