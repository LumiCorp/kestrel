## Required integration change

Read assistant output from `result.assistantText`. Do not parse a display message from `finalizedPayload`.

```ts
const terminal = await agent.run(input, context);
if (terminal.payload.result.assistantText !== null) {
  renderAssistant(terminal.payload.result.assistantText);
}
```

## Compatibility

Upgrade the Runtime, Protocol, SDK, Next.js, AI SDK, and Observability packages as one compatible `0.6.0` line. Rebuild consumers that depend on exported protocol types.

```bash
pnpm add @kestrel-agents/protocol@0.6.0 \
  @kestrel-agents/sdk@0.6.0 \
  @kestrel-agents/next@0.6.0 \
  @kestrel-agents/ai-sdk@0.6.0 \
  @kestrel-agents/observability@0.6.0
```
