Deploy covers the runtime boundary behind the `Workspace Copilot` example. The main job is not "host a site." It is to keep the app-to-runner trust model intact while the example moves from local development to a production-shaped setup.

This section is for teams that already have the tutorial working and now need to run it in a more serious environment.
