Kestrel helps agents carry real work across requests, restarts, people, and applications without taking control away from the humans responsible for the result.

## Choose what you want to accomplish

- [Download Kestrel Desktop](/desktop/install) to work with local projects and return to durable sessions later.
- [Join Kestrel One](/kestrel-one/getting-started) to continue shared agent work with your invited team.
- [Build your first agent](/build/building-your-first-agent) to add Kestrel to an application you own.

The [journey chooser](/start/quickstart) compares those starting points in more detail.

## Learn the foundation when you need it

Read [Why Kestrel](/start/why-kestrel) for the product problem, [Core concepts](/start/concepts) for the vocabulary used across the suite, and [Architecture](/start/architecture) when you need to understand how an application reaches the runtime.

These docs target `0.6.0`. Check [Release status](/start/release-status) before distributing an installer or publishing an integration.
