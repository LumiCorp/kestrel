The docs section should answer the questions that decide whether Kestrel Desktop and the broader Kestrel Suite are the right shape for your product. It explains the flagship Desktop experience, the companion surfaces around it, and the runtime and runner-service model that keep the suite coherent.

Start here if you need the product point of view before the implementation manual. These pages are for evaluation, orientation, and decision-making, not only for package lookup.
